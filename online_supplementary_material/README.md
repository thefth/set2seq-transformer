# Online Supplementary Material

This folder contains the online supplementary material for the paper:

**Set2Seq Transformer: Temporal and Position-Aware Set Representations for Sequential Multiple-Instance Learning**

## Contents

- `set2seq_transformer_online_supplementary_material.pdf`

The document includes:
- Formal definitions of training objectives and evaluation metrics.
- Detailed baseline implementation specifications.
- Full WikiArt-Seq2Rank dataset documentation.
- Additional experimental results and qualitative analysis.
- Mesogeos variable definitions and ablation studies.
- Limitations and broader impact discussion.
