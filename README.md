# Set2Seq Transformer: Temporal and Positional-Aware Set Representations for Sequential Multiple-Instance Learning

This repository contains the implementation for the paper titled:

**Set2Seq Transformer: Temporal and Positional-Aware Set Representations for Sequential Multiple-Instance Learning**.

- **Included**: Implementation using the publicly available **Mesogeos** dataset (provided in this repository).
- **Future Release**: The **WikiArt-Seq2Rank** dataset and the complete implementation will be made available upon paper acceptance.
